实用贴|如何利用Visual Studio Code为Markdown文件自动生成目录？

1.关于 Visual Studio Code 2. 如何使用 VS Code + TOC 扩展为 Markdown 文件自动生成目录呢？ 在使用Markdown写文档时，经常需要给单个文档生成目录。 那么，如何为Markdown文件（即.md格式的文件）自动生成目录呢？下面给大家介绍一种基于Visual Studio Code 软件的方法： - Visual Studio Code + Markdown TOC 扩展 1.关于 Visual Studio Code Visual Studio Code(VS Code)是一个由微软开发的，同时支持Windows/Linux和macOS操作系统的开源文本编辑器。它支持调试，内置了Git版本控制功能，同时也具有开发环境功能，例如代码不全、代码片段、代码重构等。该编辑器支持用户个性化配置，例如改变主题颜色、代码块显示颜色等友好功能，同时还在编辑器中内置了扩展程序管理的功能，通过安装扩展程序，Code 可以支持更多新的语言、界面主题、测试器，以及更多的增值服务，这也是该编辑器的一大亮点，被称为”宇宙最强编辑器”。 2. 如何使用 VS Code + TOC 扩展为 Markdown 文件自动生成目录呢？ 下载和安装Visual Studio Code 下载地址：Visual Studio Code 安装成功后，双击图标打开，找到扩展程序的图标，接着在应用商店中搜索Markdown TOC，选择一个版本，点击安装，如图： 点击菜单栏的文件 -> 打开，打开需要生成目录的 .md 格式的文件。 将光标移至需要插入目录的位置，右键单击 Markdown TOC: Insert/Update [^M T]，目录即自动插入。 - 单击右上角的预览图标，可查看目录的显示效果。
